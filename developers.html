Developers

To get familiar with the code, see the full source documentation, or browse the code on GitHub.

The build process is automated on both Windows and Linux, but has several dependencies.

Windows

  1. Install Visual Studio 2010 (Visual Studio 2005 and 2008 should also work, but are untested)
  2. Add Visual Studio's msbuild and its dependencies to your path
  3. Install Git
  4. Clone the SockIt repo into the projects directory
  5. Go to the projects/sockit directory, and run build.bat
  6. To package the plugin, add Google Chrome to your path, and run copy.bat followed by pack.bat

Linux

  1. Install dependencies:
    2. sudo apt-get install cmake libgtk2.0 doxygen git g++
  2. Clone the SockIt repo into the projects directory and go to the SockIt project directory
  3. Run make. This will build the project, generate the doxygen documentation, and package and install the plugin.
